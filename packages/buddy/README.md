@foxxytux/buddy

Metapackage for Buddy CLI and runtime packages.

Install globally:

```bash
npm i -g @foxxytux/buddy
```

This package depends on the runtime packages required by the Buddy CLI. If you prefer not to use the metapackage, install the minimal runtime:

```bash
npm i -g @foxxytux/buddy-coding-agent @foxxytux/buddy-ai @foxxytux/buddy-tui @foxxytux/buddy-agent-core
```

Usage

After installing, run the CLI with:

```bash
buddy
```
